# Lab1-prog-avanzada
## Busqueda de un numero en un Array[n][m] y conteos de ocurrencias.

Busquedas:
Dos clases que tienen el mismo objetivo de buscar un numero en un array 2d pero hecho de distintas maneras. Busqueda lineal/iterativa y busqueda Binaria.

Conteos:
Dos clases que tienen el mismo objetivo de contar las ocurrencias en un array 2d, pero buscado de manera lineal y de con un hashmap.

Para correr Busquedas:
```
mvn compile
java -cp target/classes com.busquedas.Benchmark
```

Para correr Conteos:
```
java -cp target/classes com.conteo.BenchmarkConteo
```
